  • Arithmetic
    - Arithmetic operators in JavaScript are symbols used to perform basic mathematical operations on numbers.
    (+,-,/,*,%,**)
    Here's a simple explanation with examples:
    Addition (+): Adds two numbers together.
    Example: let sum = 5 + 3; // sum will be 8
    Subtraction (-): Subtracts the second number from the first.
    Example: let difference = 10 - 4; // difference will be 6
    Multiplication (*): Multiplies two numbers.
    Example: let product = 3 * 7; // product will be 21
    Division (/): Divides the first number by the second.
    Example: let quotient = 20 / 5; // quotient will be 4
    Modulus (%): Returns the remainder when the first number is divided by the second.
    Example: let remainder = 15 % 4; // remainder will be 3
    Exponentiation (**): Returns the power of the first number.
    Example: let remainder = 3 ** 2; // Exponentiation will be 9
    These operators are fundamental for performing calculations in JavaScript, whether you're working with numbers directly or using variables that store numeric values.

    Assignment
    - Assignment operators in JavaScript are symbols used to assign values to variables and modify the values of variables. Here's a simple explanation with examples:
    (=,+=,-=,*=,/=,**=,%=)
    Assignment (=): Assigns a value to a variable.
    Example: let x = 10; // Assigns the value 10 to the variable x
    Addition Assignment (+=): Adds a value to the current value of a variable and assigns the result back to the variable.
    Example: let y = 5; y += 3; // Adds 3 to the current value of y (5 + 3) and assigns the result (8) back to y
    Subtraction Assignment (-=):
    Subtracts a value from the current value of a variable and assigns the result back to the variable.
    Example: let z = 10; z -= 2; // Subtracts 2 from the current value of z (10 - 2) and assigns the result (8) back to z
    Multiplication Assignment (*=): Multiplies the current value of a variable by a value and assigns the result back to the variable.
    Example: let a = 3; a *= 4; // Multiplies the current value of a (3) by 4 and assigns the result (12) back to a
    Division Assignment (/=): Divides the current value of a variable by a value and assigns the result back to the variable.
    Example: let b = 20; b /= 5; // Divides the current value of b (20) by 5 and assigns the result (4) back to b
    Modulus Assignment (%=): Calculates the remainder when dividing the current value of a variable by a value and assigns the result back to the variable.
    Example: let c = 15; c %= 4; // Calculates the remainder when dividing the current value of c (15) by 4 and assigns the result (3) back to c
    These operators are useful for updating the values of variables based on calculations or other operations in JavaScript.
    Comparison
    - Comparison operators in JavaScript are symbols used to compare two values and return a Boolean result (true or false) based on the comparison. Here's a simple explanation with examples:
    (<,>,==, <=,>=,===,!==,!=)
    Equal to (==): Compares two values to check if they are equal.
    Example: 5 == 5 // Returns true because 5 is equal to 5
    Not equal to (!=): Compares two values to check if they are not equal.
    Example: 5 != 3 // Returns true because 5 is not equal to 3
    Strict equal to (===): Compares two values to check if they are strictly equal in value and type.
    Example: '5' === 5 // Returns false because the string '5' is not strictly equal to the number 5
    Strict not equal to (!==): Compares two values to check if they are not strictly equal in value or type.
    Example: '5' !== 5 // Returns true because the string '5' is not strictly equal to the number 5
    Greater than (>): Checks if the first value is greater than the second value.
    Example: 10 > 5 // Returns true because 10 is greater than 5
    Less than (<): Checks if the first value is less than the second value.
    Example: 3 < 8 // Returns true because 3 is less than 8
    Greater than or equal to (>=): Checks if the first value is greater than or equal to the second value.
    Example: 8 >= 8 // Returns true because 8 is equal to 8
    Less than or equal to (<=): Checks if the first value is less than or equal to the second value.
    Example: 5 <= 3 // Returns false because 5 is not less than or equal to 3
    These operators are essential for comparing values and controlling the flow of execution in JavaScript programs.
    String
    - There is no such thing as "string operators" in JavaScript. However, there are various operations that can be performed on strings using methods and operators. Here are some common operations related to strings in JavaScript:
    Concatenation (+): Combines two or more strings together.
    Example: let fullName = firstName + ' ' + lastName;
    String Length Property (length): Returns the length of a string.
    Example: let messageLength = message.length;
    Accessing Characters (Bracket Notation or charAt()): Allows you to access individual characters in a string.
    Example: let firstChar = str[0]; or let firstChar = str.charAt(0);
    Substring (substring()): Extracts a portion of a string.
    Example: let subStr = str.substring(startIndex, endIndex);
    String Methods:
    - JavaScript provides numerous built-in methods for manipulating strings, such as toUpperCase(), toLowerCase(), indexOf(), lastIndexOf(), slice(), split(), replace(), and many more.
    - These are some of the common operations you can perform on strings in JavaScript. They allow you to manipulate and work with textual data effectively in your programs.

  • Logical
    - Logical operators in JavaScript are used to combine or manipulate Boolean values. Here are the main logical operators along with their explanations:
    (&&,||,!)
    Logical AND (&&): Returns true if both operands are true; otherwise, it returns false.
    Example: true && true returns true, but true && false or false && false returns false.
    Logical OR (||): Returns true if at least one of the operands is true; returns false if both operands are false.
    Example: true || false returns true, but false || false returns false.
    Logical NOT (!): Reverses the Boolean value of its operand; if the operand is true, it returns false, and if the operand is false, it returns true.
    Example: !true returns false, and !false returns true.
    -Logical operators are often used to make decisions in conditional statements, control the flow of execution in programs, and perform logical operations on Boolean values.

    Bitwise
    Bitwise operators in JavaScript manipulate the binary representation of numbers at the bit level. Here are the bitwise operators along with their explanations:
    (++,--)
    Bitwise AND (&): Performs a bitwise AND operation on each pair of corresponding bits. If both bits are 1, the result is 1; otherwise, it's 0.
    Example: 5 & 3 returns 1 because the binary representation of 5 (101) AND 3 (011) yields 001.
    Bitwise OR (|): Performs a bitwise OR operation on each pair of corresponding bits. If at least one bit is 1, the result is 1; otherwise, it's 0.
    Example: 5 | 3 returns 7 because the binary representation of 5 (101) OR 3 (011) yields 111.
    Bitwise XOR (^): Performs a bitwise XOR (exclusive OR) operation on each pair of corresponding bits. If the bits are different, the result is 1; otherwise, it's 0.
    Example: 5 ^ 3 returns 6 because the binary representation of 5 (101) XOR 3 (011) yields 110.
    Bitwise NOT (~): Performs a bitwise NOT operation, which inverts all the bits of its operand. It returns the two's complement of the number.
    Example: ~5 returns -6 because the bitwise NOT of 5 (101) yields 11111111111111111111111111111010, which is -6 in two's complement representation.
    Left Shift (<<): Shifts the bits of the left operand to the left by a specified number of positions, filling in zeros from the right.
    Example: 5 << 1 returns 10 because the binary representation of 5 (101) shifted left by 1 position yields 1010, which is 10 in decimal.
    Right Shift (>>): Shifts the bits of the left operand to the right by a specified number of positions, filling in the leftmost bits with the sign bit (0 for positive numbers and 1 for negative numbers).
    Example: 5 >> 1 returns 2 because the binary representation of 5 (101) shifted right by 1 position yields 10, which is 2 in decimal.
    Unsigned Right Shift (>>>): Similar to the right shift operator (>>), but it always fills in the leftmost bits with zeros, regardless of the sign of the number.
    Example: 5 >>> 1 returns 2 because the binary representation of 5 (101) shifted right by 1 position yields 10, which is 2 in decimal.
    - Bitwise operators are typically used in low-level programming, such as optimizing algorithms, working with binary data, and implementing certain bitwise operations.
    Ternary
    - The ternary operator, also known as the conditional operator, is a concise way to write conditional statements in JavaScript. It takes three operands and evaluates a condition, returning one of two expressions based on whether the condition is true or false. Here's the syntax and an example:
    Syntax:condition ? expression1 : expression2
    Explanation:
    - If the condition evaluates to true, the expression before the colon (:) is returned.
    - If the condition evaluates to false, the expression after the colon (:) is returned.
    Example: let age = 20;
    let message = (age >= 18) ? 'You are an adult' : 'You are a minor';
    console.log(message); // Output: 'You are an adult'
    In this example:
    - If the condition age >= 18 is true, the expression 'You are an adult' is returned.
    - If the condition is false, the expression 'You are a minor' is returned.
    Ternary operators are often used for inline conditional assignments, making code more concise and readable. However, they should be used judiciously to maintain code clarity.
    Unary
    - Unary operators in JavaScript work on a single operand and perform various operations. Let’s explore them with simple examples:
    Unary Plus (+) Operator: The unary plus converts an operand into a number, if possible. It’s commonly used to ensure numerical operations on variables that may contain numeric strings.
    Example:
    let str1 = "12";
    let num = +str1; // Converts the string "12" to the number 12
    console.log(num); // Output: 12
    Unary Minus (-) Operator: The unary negation (-) operator converts its operand to a negative number if it isn’t already negative.
    Example:
    let str2 = -"Geeks"; // Negates the value (results in NaN)
    console.log(str2); // Output: NaN
    Unary Increment (++) Operator: Adds 1 to its operand’s value and evaluates to the updated value. Can be used as a postfix or prefix operator.
    Example (postfix)
    let num = 12;
    let numPostfix = num++; // First uses num, then increments it
    console.log(numPostfix); // Output: 12
    console.log(num); // Output: 13
    Unary Decrement (--) Operator: Subtracts 1 from its operand’s value and evaluates to the updated value. Also used as a postfix or prefix operator.
    Example (prefix)
    let num1 = 10;
    let num1Prefix = ++num1; // Increments num1 first, then uses the updated value
    console.log(num1Prefix); // Output: 11
    console.log(num1); // Output: 11
    Remember, unary operators are like little tools that help you manipulate values in JavaScript!
